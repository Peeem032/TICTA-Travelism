# Project Overview

This project implements a **Proof of Concept (POC)** for our `Decision Model`, which will serve as the core algorithm for our production system.

---

# Usage

To run this project, install the required dependencies from `requirements.txt`:

```bash
pip install -r requirements.txt
```
💡 Note: I fixed the command from `-f requirements.txt` → `-r requirements.txt`, which is the standard way to install Python dependencies.
